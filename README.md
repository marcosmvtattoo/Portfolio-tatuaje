# Mi Web - Proyecto Base Portafolio

Este es un proyecto base para realizar una web tipo portafolio. Incluye las herramientas y dependencias necesarias para crear una aplicación moderna y profesional.

## Dependencias principales

- **React** - Biblioteca para construir interfaces de usuario
- **Tailwind CSS** - Framework de CSS para estilos rápidos y modernos
- **Shadcn/ui** - Colección de componentes UI reutilizables y accesibles
- **React Router** - Navegación y enrutamiento para aplicaciones React
- **Vite** - Herramienta de construcción y desarrollo rápido

## Comandos principales

- `npm install`: Instala todas las dependencias necesarias que están listadas en `package.json`.

- `npm run dev`: Ejecuta el servidor de desarrollo, permitiendo ver y probar la aplicación en modo interactivo en local.

- `npm run build`: Genera una versión optimizada y lista para producción de la aplicación en la carpeta `build`.

## Uso

1. Clona este repositorio.

2. Ejecuta `npm install` para instalar las dependencias.

3. Ejecuta `npm run dev` para desarrollar y previsualizar en entorno local.

4. Cuando quieras publicar la versión final, ejecuta `npm run build`.
